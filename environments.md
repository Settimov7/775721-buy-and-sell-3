## Api server

`API_SERVER_PORT` - порт api сервера;  

## СУБД

`DB_HOST` - хост;  
`DB_PORT` - порт;  
`DB_USER_NAME` - имя пользователя;  
`DB_NAME` - название БД;  
`DB_PASSWORD` - пароль;  

## СУБД для тестов

`TEST_DB_HOST` - хост;  
`TEST_DB_PORT` - порт;  
`TEST_DB_USER_NAME` - имя пользователя;  
`TEST_DB_NAME` - название БД;  
`TEST_DB_PASSWORD` - пароль;
